Telugu Stopwords

A comprehensive Python library for Telugu stopwords, designed for NLP tasks like hate speech detection, sentiment analysis, and topic modeling.

Features

  • Extensive Telugu stopwords list in Telugu script and English transliteration
  • Add/remove custom stopwords
  • JSON save/load support
  • Function to clean stopwords from text
  • Handles code-mixed Telugu-English text
  • Simple imports: tsw for TeluguStopWords and rmtsw for remove_stopwords

Installation

pip install telugu-stopwords

From GitHub:

pip install git+https://github.com/PavanYellathakota/telugu-stopwords.git

Development mode:


cd telugu-stopwords
pip install -e .

Usage

from telugu_stopwords import tsw, rmtsw

telugu_sw = tsw()

# Get Telugu script stopwords
print(telugu_sw.get_stopwords(script="telugu")[:5])

# Remove stopwords
text = "నేను ఈ రోజు చాలా సంతోషంగా ఉన్నాను కాబట్టి బయటకు వెళ్లాలనుకుంటున్నాను"
print(rmtsw(text, script="telugu"))

# Add custom stopwords
telugu_sw.add_stopwords(["రోజు"], script="telugu")

# Save to JSON
telugu_sw.save_to_json("custom_stopwords.json")

Notes

  • tsw is alias for the class TeluguStopWords
  • rmtsw is alias for the function remove_stopwords
  • Supports both Telugu (script="telugu") and English transliterated stopwords

Development Setup

  1. Clone the repo: 
    git clone https://github.com/PavanYellathakota/telugu-stopwords.git
  2. Update pyproject.toml with your author info
  3. Install dependencies: 
    
pip install pytest
pip install -e .
  4. Run tests: 
    pytest tests/
